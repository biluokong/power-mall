当前只实现了视频中讲解的功能。

- 前端项目：`mall4v`
- 后端项目：`power-all`
- 小程序端：`mall4m`



**node.js版本：16.15.0**，可以使用nvm工具切换到该版本后运行，否则可能会运行失败。

~~~bash
# 如果没有下载nodejs-16.15.0
nvm install 16.15.0
nvm use 16.15.0
# 设置淘宝npm源
npm config set registry https://registry.npmmirror.com/
# 下载依赖和运行项目（需要在mall4v文件夹下）
npm install
npm run dev
~~~
